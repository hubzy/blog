模型生产规范

一、 模型的定义

1. 颜色定义

1.1. 基本色调（主色调）

​ 模型材质色调主要引用玲珑色系。在单物体模型时可使用玲珑蓝色材质作为主色调，搭配合视觉界面进行调整。

​ 涉及到透明机身展示部分，可在基础材质球上添加透明属性，透明属性中折射率预设选用‘塑料（PET）’即可。

1.2. 特殊部分色调（核心部件色调）

​ 特殊部件材质针对需要监测的部件使用，具体使用的颜色可根据实际效果进行替换。

​ 次要部件可使用灰色材质，根据不同层级灰度选择不同的灰色材质。

2. 状态定义

2.1. hover描边效果

​ 当鼠标hover到某个部件时，此部件显示描边效果

2.2. active-同上保持描边

2.3. 错误类型状态（材质球）

2.4. 警告类型状态（材质球）

2.5. 成功类型状态（材质球）

3. 交互方式定义

3.1. 默认模型动画定义

3.2. 主动介入交互方式

3.3. 模型加载提示

3.4. 错误提示

3.5. 模型基本操作

二、 参数定义

参数定义背景：在不同的参数下，模型效果的呈现的效果不统一，为了减少设计与研发调试带来的不必要的麻烦以及提高模型视觉效果还原度，所以制定统一的配置参数。后续设计会依据统一的参数进行模型制作与设计。

1. canvas背景颜色

2. Renderer-渲染器设置

3. OrbitControls-范围控制

4. 灯光设置

三、 生产流程

1. 模型来源

1.1. 产品提供或购买模型（优先考虑XXX格式）

1.2. 自己寻找模型或制作模型

2. 模型编辑

2.1. glb、gltf、格式可用blender编辑或转为c4d可编辑格式

2.2. obj、3ds、fbx、.STEP格式可用C4D直接编辑

2.3. 使用同一套材质球

2.4. 格式转换选FBX为中间间

3. 输出/交付

3.1. 模型处预处理后保留源文件（本地服务器盘中）

3.2. 在副本上对模型进行优化，图层合并、删减面、拆分编组、编组规范命名（代确认）、坐标轴归原点、默认角度。

3.3. 导出.glb格式（同一导出设置下.glb格式相对比.gltf要小）

1. 导出glTF(*.gltf/*.glb)
2. 设置文件格式选GLB(Binary)
3. 比例1厘米根据合适的比例缩放
4. 仅勾选当前帧、翻转Z，其余都不选
5. 上传模型库`.glb` 模型文件（上传方式待更新）
